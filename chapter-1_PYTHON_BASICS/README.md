Question 1: Which of the following are operators, and which are values?
```bash
*
'hello'
-88.8
-
/
+
5
```

Answer: Here 'hello', -88.8 and 5 is values and rest of it is operators.
"*" it is call multiplayer

"-" it is called subtrctor.

"/" it is called divider

"+" it is called adder.
